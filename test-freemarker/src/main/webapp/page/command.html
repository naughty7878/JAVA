List常用指令

<#assign myList=[2,3,4,5,1,8,9,7] />
排序指令sort
    <#list myList?sort as item >
  • ${item_index } - ${item }

导至输出reverse
    <#list myList?sort?reverse as item >
  • ${item_index } - ${item }

大小
  • myList?size - ${myList?size }
  • myList[3] - ${myList[3] }
