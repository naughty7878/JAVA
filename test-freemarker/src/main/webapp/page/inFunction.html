内建函数

1、字符串内建函数

<#assign myList=[2,3,4,5,1,8,9,7] />
split
    <#list 'a|b|c|d'?split('|') as item >
  • ${item }

字符串转日期
    <#assign var1 = "01/03/2017"?date("MM/dd/yyyy") /> <#assign var2 = "15:05:30"?time("HH:mm:ss") /> <#assign var3 = "2019-07-22 03:05 PM"?datetime("yyyy-MM-dd HH:mm") />
  • ${var1 }
  • ${var2 }
  • ${var3 }

2、数字内建函数

<#assign numVar1=3.1415926 />
3.1415926
  • ${numVar1?string("0.##") }
  • ${numVar1?round }

3、List内建函数

<#assign myList1=[2,3,4,5,1,8,6,9,7] />
chunk
  • ${myList1?chunk(3)?size }
    • <#list myList1?chunk(3)?last as item >
  • ${item }

4、其他内建函数

<#assign sVar="hello" />
hello
  • ${sVar?is_number?string('yes', 'no') }
  • ${sVar?has_content?string('yes', 'no') }
  • ${"1" + "2" ? eval }
  • ${vDefault!"sss" }
