模板中使用macro指令定义

1、macro:宏指令


无参数的macro
    <#macro test>
  • 我是无参的macro
    • <@test />

有参数的macro
    <#macro test param1 param2>
  • 我是有参的macro param1 = ${param1}, param2 = ${param2}
    • <@test param1="java" param2="python" />

有默认值参数的macro
    <#macro test param1 param2="def">
  • 我是有默认值参数的macro param1 = ${param1}, param2 = ${param2}
    • <@test param1="java" />

有多个参数的macro
    <#macro test param1 param2="def" paramExt... >
  • 有多个参数的macro param1 = ${param1}, param2 = ${param2}
  • ${paramExt["param3"]}
    • <@test param1="java" param2="python" param3="html"/>

2、nested


嵌套
    <#macro test param1="java">
  • ${param1 }
    • 我是无参的macro
    <#nested param1, "我的nested 参数"> <@test param1="java1"; loopVar1, loopVar2> hello ${loopVar1}, ${loopVar2} <@test param1="java2"; loopVar1, loopVar2> hello ${loopVar1}, ${loopVar2}

2、函数


函数调用
    <#function doAdd param1 param2> <#return param1+param2 >
  • ${doAdd(20, 30) }
