字符串

字符串常用内置函数

    <#assign a='hello'/> <#assign b='world'/>
    "+" 加号连接
  • a + b = ${a + b }

    • substring 截取
  • (a + b)?substring(5,8) = ${(a + b)?substring(5,8) }

    • length 长度
  • (a + b)?length = ${(a + b)?length }

    • upper_case 大写
  • (a + b)?upper_case = ${(a + b)?upper_case }

    • lower_case 小写
  • (a + b)?lower_case = ${(a + b)?lower_case }

    • index_of 位置
  • (a + b)?index_of('w') = ${(a + b)?index_of('w') }

    • replace 替换
  • (a + b)?replace('w', 'W' ) = ${(a + b)?replace('w', 'W') }

    • contains 包含
  • (a + b)?contains('w') = ${(a + b)?contains('w') }
