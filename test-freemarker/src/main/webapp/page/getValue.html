逻辑

1、取基本值
  • 整型：${intValue }
  • 字符串：${stringValue }
  • double型：${doubleValue }
  • 布尔型：${booleanValue }
  • 时间：${dateValue?string('yyyy-MM-dd') }
  • null值：${nullValue}
2、运算
  • 加法：${intValue + 100}
3、集合
    List集合 <#list list as item>
  • ${item}
    • Map集合 <#list map?keys as key>
  • ${key}:${map[key]}
